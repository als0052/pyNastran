All dynamic control cards are defined in this file. This includes:

  • FREQ
  • FREQ1
  • FREQ2
  • FREQ3 (not implemented)
  • FREQ4
  • FREQ5 (not implemented)
  • NLPCI
  • NLPARM
  • TSTEP
  • TSTEP1
  • TSTEPNL
  • ROTORG
  • ROTORD
  • TIC
  • TF

All cards are BaseCard objects.
