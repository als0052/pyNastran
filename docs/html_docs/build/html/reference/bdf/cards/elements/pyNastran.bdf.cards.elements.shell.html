All shell elements are defined in this file. This includes:

  • CTRIA3
  • CTRIA6
  • CSHEAR
  • CQUAD
  • CQUAD4
  • CQUAD8
  • CQUADR
  • CPLTSN3
  • CPLSTN4
  • CPLSTN6
  • CPLSTN8
  • SNORM

All tris are TriShell, ShellElement, and Element objects. All quads are QuadShell, ShellElement, and Element objects.
