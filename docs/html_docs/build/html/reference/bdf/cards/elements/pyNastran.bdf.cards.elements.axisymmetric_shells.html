All axisymmetric shell elements are defined in this file. This includes:
  • CTRAX3
  • CTRAX6
  • CTRIAX
  • CTRIAX6
  • CQUADX
  • CQUADX4
  • CQUADX8

All tris are TriShell, ShellElement, and Element objects. All quads are QuadShell, ShellElement, and Element objects.
