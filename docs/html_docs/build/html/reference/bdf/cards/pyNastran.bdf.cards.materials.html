All material cards are defined in this file. This includes:

  • CREEP
  • MAT1 (isotropic solid/shell)
  • MAT2 (anisotropic)
  • MAT3 (linear orthotropic)
  • MAT4 (thermal)
  • MAT5 (thermal)
  • MAT8 (orthotropic shell)
  • MAT9 (anisotropic solid)
  • MAT10 (fluid element)
  • MATHP (hyperelastic)
  • MATHE (hyperelastic)

All cards are Material objects.
